# Cloudy Weather API

A Python + Django backend project that works with Docker and PostgreSQL.  
This project allows you to fetch weather data for a city via an API and store it in the database.

---

## Technologies

- Python 3.13  
- Django 6.x  
- Django REST Framework  
- Docker & Docker Compose  
- PostgreSQL 17  
- Redis (for caching)  
- Requests (for API calls)
